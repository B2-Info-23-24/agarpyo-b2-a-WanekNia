# Agarpyo

Generic Aragio like with Python and Pygame

## Clone the project
```
git clone https://github.com/B2-Info-23-24/agarpyo-b2-a-WanekNia.git
```

## Installation

you need to have 'Anaconda' in your system : 'https://www.anaconda.com/download'

in Anaconda prompt :
```bash
conda create --name agarpyo python=3.8
conda activate agarpyo
pip install pygame
```

## Usage

Directly in the anaconda prompt
```
conda activate agarpyo
cd [clone directory]
python menu.py
```
